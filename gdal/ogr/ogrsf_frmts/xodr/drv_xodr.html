OpenDRIVE (XODR)

The OpenDRIVE driver provides read access for OpenDRIVE XML documents in version 1.4. The implemented schema is backwards-compatible but support for versions lower than 1.4 is not guaranteed.

Requirements

  • xodr
  • odrSpiral
  • Xerces-c 3.1
  • GDAL >= 2.x

Dataset Creation Options

None.

Layer Creation Options

None.

Building Notes

Detailed instruction on how to build the OpenDRIVE driver is provided in the repository's README.
